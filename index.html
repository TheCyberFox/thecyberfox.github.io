Project Highlights

Projects I worked on during my time at CBU

Path Finder/Maze Solver

This path finder/maze solver uses breadth first and depth first search to determine the fastest possible route to a given location. The starting location is on the upper left hand corner and the ending location is on the bottom right hand corner.

N-Queens Problem Solver

The following project set out to solve the N-Queens problem. The N-Queens problem is a problem designed to place as many queens on a chessboard without any queen being able to capture another in one turn. This project also took into consider placing barrier on the board.

Tibetan Bible Capstone

The Tibetan Bible Capstone project was a website that we had to take from an unresponsive website based in HTML4 and make it into a responsive website that is mobile friendly in HTML 5. We also set out to create a text to speech feature which required that we had to develop an OCR (optical character recognition) software that scans images and pdfs for text characters and based on those images, it outputs text to a certain GUI.
